## Multi-step form

Реализация формы с несколькими шагами.

Шаг первый: три обязательных для заполнения поля - Имя, почта и номер телефона.

Шаг второй: выбор подписки на месяц или год и одного из трех планов - Arcade, Advanced, Pro.

Шаг третий: выбор дополнений к плану.

Шаг четвертый: отображение выбранного плана и тип подписки, а также все выбранные дополнения и общую стоимость. Нажатие на надпись `change` под выбранным планом отправляет на 2 страницу.

Cсылка на макет: https://www.figma.com/file/9WSKUdq7rIfOysWQ7nS7d5/multi-step-form?node-id=0%3A1170&t=6F0rrr8nAxFy5G3a-1

## Как развернуть приложение

Установить все пакеты:

```bash
    npm i

```

Запустить приложение:

```bash
    npm start

```
